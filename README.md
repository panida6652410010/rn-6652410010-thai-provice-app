## การติดตั้งและใช้งาน (Installation)

1. คลอนโปรเจกต์ (Clone Project)
   ```bash
   git clone [your-repository-url]
   ```

2. ติดตั้ง dependencies
   ```bash
   npm install
   ```

3. รันโปรเจกต์
   ```bash
   npx expo start
   ```
